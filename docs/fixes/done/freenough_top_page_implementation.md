# 指示書: FREENOUGHトップページ実装

作成日: 2026-07-19
対象: `freenough-main`プロジェクトの`app/page.tsx`・`app/layout.tsx`
種別: 新規実装

---

## 1. 背景

`freenough-main`はFreenoughブランドのトップページ用プロジェクト。現状は`create-next-app`のデフォルト画面のまま。以下のブランドコンセプトに基づき、トップページを実装する。

- ブランド名の由来:Free + Enough(「多くを望まない」ではなく「自分にとっての"足りる"を知る」)
- 位置づけ:今のトップページは「サービス一覧」ではなく「ブランドの思想紹介ページ」として割り切る。現時点で提供中のサービスはLifeCompassのみ
- 役割分担:ブランド(Freenoughトップ)は思想を語り、サービス(LifeCompassのLP)は機能を語る。そのため、このページではLifeCompassの機能説明・サンプルチャート・特徴一覧などは書かない
- Heroにイメージ画像は入れない(テキスト中心のシンプルな構成)
- ブログ・Noteの記事プレビューはトップページ本文には置かない(ヘッダーナビのリンクのみ)

---

## 2. 実装内容

### 2-1. ロゴ(ヘッダー)

- 円形のコンパスアイコンは使わない
- 「FREENOUGH」のテキストロゴのみ。"FREE"と"ENOUGH"が共有する2つ目のE(4文字目)にだけ下線を入れる
  - 例:`FRE` + `<span className="underline decoration-2 underline-offset-4 decoration-[#3F9C6D]">E</span>` + `NOUGH`
- ヘッダー右側にナビゲーション:LifeCompass / ブログ / Note(それぞれリンク先は下記3を参照)

### 2-2. Hero

3行に分けて表示する(改行の位置が意味的にも自然になるよう調整済み):

```
あなたにとっての
「足りる」を、
数字で描く。
```

- 文字サイズは大きめ(目安:`text-4xl sm:text-5xl`程度、太字)
- 画像・イラストは入れない

### 2-3. ブランドストーリー

見出し:「なぜ「Freenough」なのか」(控えめなラベルとして、小さめ・グレー系の文字色)

本文:

```
人生に必要なお金も、理想の働き方も、人それぞれです。

大切なのは、誰かの正解を追いかけることではなく、
自分にとって「足りる(Enough)」を知ること。

Freenoughは、その「足りる」を、
体験談ではなく、数字で描く場所です。
```

- 中央寄せ、行間広め、最大幅は480px程度に制限して読みやすくする

### 2-4. 現在提供中セクション

- ラベル:「現在提供中」(小さく、グレー、レタースペース広め)
- カード(枠線・角丸、中央寄せのコンテンツ):
  - **アイコン+「LifeCompass」の文字を横並びで配置**(アイコンは`public/images/compass_logo.png`を使用。前回モックのSVG代替ではなく、実際のロゴ画像ファイルを使うこと)
  - 一言のみ:「あなたの「足りる」を、シミュレーションで確かめる。」(機能説明・特徴一覧は書かない)
  - ボタン:「LifeCompassを見る →」
    - **リンク先はLifeCompassのLP**(現状のVercel URL、または独自ドメイン接続後はそちらに差し替え。シミュレーター本体やブログへの直リンクにはしないこと)

### 2-5. フッター

- 中央寄せ、小さい文字で:「© FREENOUGH — Design Your Enough.」

### 2-6. やらないこと(明示的に対象外)

- ブログ・Noteの記事プレビューをトップページ本文中に置かない(ヘッダーナビのリンクのみで十分)
- LifeCompass以外のサービス(hojin・AI等)のカードは一切置かない(名前も出さない)
- Heroにイメージ画像・イラストは入れない

---

## 3. アセット

- `public/images/compass_logo.png`を配置すること(添付のロゴ画像ファイルをKENZOから受け取って配置する。まだ配置されていない場合は、その旨を報告してKENZOに確認すること)

---

## 4. メタデータ(`app/layout.tsx`)

- `title`:「FREENOUGH — Design Your Enough.」程度
- `description`:「あなたにとっての「足りる」を、数字で描く。FREENOUGHは、シミュレーションとデータを通じて、人生のお金の意思決定を支援するブランドです。」
- OGP(`openGraph`)は、今回は仮設定でよい(専用のOGP画像がまだないため、当面はデフォルトのまま。今後Freenough用のOGP画像を別途用意する想定)

---

## 5. 受け入れ基準

- [ ] Heroが3行(あなたにとっての/「足りる」を、/数字で描く。)で表示され、画像は入っていない
- [ ] ヘッダーロゴが「FRE**E**NOUGH」のテキストのみで、2つ目のEに下線が入っている(丸いアイコンは表示されない)
- [ ] LifeCompassカードに実際のロゴ画像(`compass_logo.png`)が表示され、機能説明文が書かれていない(一言のみ)
- [ ] 「LifeCompassを見る」ボタンがLifeCompassのLPへリンクしている(シミュレーター本体やブログへの直リンクになっていない)
- [ ] トップページ本文中にブログ・Noteの記事プレビューが存在しない
- [ ] `npm run build`が型エラーなしで通る
