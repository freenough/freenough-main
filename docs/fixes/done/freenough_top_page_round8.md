# 指示書: FREENOUGHトップページ 調整(第8ラウンド)

作成日: 2026-07-19
対象: `freenough-main`の`app/layout.tsx`(または`globals.css`)
種別: 追加修正

---

## 背景

コンテンツ量が少ない・画面が高い場合、フッターより下にページの余白が残り、そこがデフォルトの白色になっている。ヘッダー・Hero・ブランドストーリー・LifeCompassブロック・フッターは、それぞれ個別に背景色(`bg-white`または`bg-slate-50`)を指定済みのため、フッターより下の「どの要素の背景指定も及んでいない領域」だけが白く見えている状態。

## 対応内容

- `body`(または`html`)要素に`bg-slate-50`を指定する
- 実装後、ヘッダー・Hero・ブランドストーリーの帯・LifeCompassブロック・フッター、それぞれの背景色が意図せず変化していないか確認すること(各セクションは個別に背景色を明示指定しているため、通常は影響を受けないはずだが、念のため目視確認する)
- フッターより下の余白部分が、フッターと同じ`bg-slate-50`色になり、「白の継ぎ目」が見えなくなっていることを確認する

---

## 受け入れ基準

- [ ] フッターより下の余白部分が`bg-slate-50`になっている
- [ ] ヘッダー・Hero・ブランドストーリー・LifeCompassブロック・フッターの各背景色に意図しない変化がない
- [ ] `npm run build`が型エラーなしで通る
