# 指示書: FREENOUGHトップページ 微調整

作成日: 2026-07-19
対象: `freenough-main`の`app/page.tsx`
種別: 追加修正
補足: `freenough_top_page_implementation.md`の完了報告を踏まえた調整

---

## 1. ヘッダーロゴの開始位置を修正

現状、「FREENOUGH」ロゴの開始位置が画面左端から一定の余白を持って始まっている(LifeCompass側の「LifeCompass」ロゴが画面左端ギリギリから始まっているのに対し、Freenough側はやや内側から始まっている)。

- ヘッダーのコンテナに、意図しない`max-w-*`やセンタリング(`mx-auto`で中央寄せされた狭いコンテナ)がかかっていないか確認する
- LifeCompass側のヘッダー実装(`px-4`や`px-6`など、画面端からの余白の取り方)と揃え、左端の開始位置を一致させる
- 具体的には、ヘッダーの外側コンテナを`w-full`+左右均等の`px-*`のみとし、内部に不要な`max-w`制約が付いていればそれを外す(またはLifeCompass側と同じ`max-w`値に統一する)

## 2. Heroを2行に変更

現状3行になっているものを、以下の2行構成に変更する:

```
あなたにとっての「足りる」を、
数字で描く。
```

## 3. 「現在提供中」ラベルを削除

- カードの上にある「現在提供中」というラベルテキストを削除する
- カード自体(ロゴ+LifeCompass名+一言+ボタン)はそのまま残す

## 3-2. 「なぜ「Freenough」なのか」ラベルを削除

- ブランドストーリー本文の上にある「なぜ「Freenough」なのか」という見出しラベルを削除する
- 本文(「人生に必要なお金も、理想の働き方も...」以降)はそのまま残す

## 4. 「LifeCompassを見る →」ボタンのスタイル変更

現状、テキストリンクのような見た目になっている。これを、LifeCompassのLPにある「今すぐシミュレーションする →」ボタンと同じスタイル(紺色背景・白文字・角丸の塗りつぶしボタン)に変更する。

- 背景色:LifeCompass側のボタンと同じ紺色(既存のブランドカラー、例:`#0F2A4A`または該当するTailwindクラス)
- 文字色:白
- 形状:角丸(LifeCompass側のボタンと同じ`rounded-*`の値)
- パディング・フォントサイズもLifeCompass側のボタンに合わせる

## 5. ページ全体の余白・間延び感の調整

現状、デスクトップの広い画面幅で見ると、中央の文字列だけが浮いていて、左右に広い余白が残り「間延びした」印象になっている。以下の対応で調整する:

- ページ全体(Hero〜ブランドストーリー〜現在提供中カード)を`max-w-2xl`程度のコンテナに収め、中央に配置する
- ブランドストーリーのセクションに、他のセクションよりわずかに濃い背景色(例:`bg-slate-50`)を敷き、セクションごとの区切りを視覚的なリズムとして出す(情報量は増やさない、背景色の切り替えのみ)
- 各セクション間の縦方向の余白(padding)を、窮屈にならない程度に見直す(現状のバランスを大きく壊さない範囲で微調整)
