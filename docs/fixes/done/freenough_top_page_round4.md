# 指示書: FREENOUGHトップページ 調整(第4ラウンド)

作成日: 2026-07-19
対象: `freenough-main`の`app/page.tsx`
種別: 追加修正

---

## 1. Heroの折り返し修正

現状、Heroが`max-w-2xl`のコンテナに収まっているため、1行目「あなたにとっての「足りる」を、」が意図せず2行に折り返され、結果的に3行表示になってしまっている。

- Hero部分だけ、本文(ブランドストーリー)より広いコンテナ幅にする(例:`max-w-4xl`程度、または実際に1行目のテキストが折り返さない幅を確認して調整する)
- 2行構成(1行目:あなたにとっての「足りる」を、/2行目:数字で描く。)を維持しつつ、**1行目が意図せず途中で折り返されないこと**を実際の画面で確認する
- ブランドストーリー本文の幅(`max-w-2xl`)はそのまま変更しない(Heroのコンテナ幅だけ個別に広げる)

## 2. セクション間の余白を詰める

現状、Hero・ブランドストーリー・LifeCompassカードの間の余白が広く、ページ全体がファーストビュー(スクロールなしの1画面)に収まらず、フッターが見えない状態になっている。

- Hero〜ブランドストーリー〜LifeCompassカードの各セクション間の縦方向の余白(padding/margin)を詰める
- 目安として、一般的なデスクトップ画面の高さ(1080px程度)で、フッターまで含めて1画面に収まることを目指す
- ただし詰めすぎて窮屈にならないよう、実際の画面で見た目のバランスを確認しながら調整すること

## 3. フッターからGitHubを削除

- フッターのリンク一覧から「GitHub」を削除する
- 残すリンク:About / X / Note
- レイアウト(横並び中央寄せ)はそのまま維持

---

- [ ] フッターのリンクがAbout/X/Noteの3つのみになっている(GitHubが削除されている)
