作成日: 2026-07-19
対象: `freenough-main`の`app/page.tsx`
種別: 追加修正

---

## 1. フッターの背景色

- フッターの背景色を`bg-slate-50`にする
- 参考:`lifecompass-next`側のフッターも`bg-slate-50`(実際のコードで確認済み)。これに揃える

## 2. LifeCompassカードの枠を完全に外す(箱自体をなくす)

現状、カードは`rounded-2xl border border-black/10`のような枠線付きの箱になっている。

- 枠線・角丸・背景色・影、いずれも付けない。**箱自体をなくし、ロゴ・テキスト・ボタンが直接ページの背景の上に浮かんでいる状態**にする
- 参考:LifeCompass側の末尾CTAセクション(「まず、自分の数字を入れてみる。」)がまさにこの形式で、枠線も影もない。これはLifeCompass側が「複数の項目を並べるグリッド」ではなく「単独の最終CTA」であるため。Freenoughトップのこのブロックも、LifeCompassという1つのサービスだけを扱う単独のCTAなので、同じ考え方を適用する
- 佐々木さん・田中さんのような、複数項目を並べるグリッドカード(枠線+影で項目同士を区切る必要があるもの)とは役割が異なるため、そちらのスタイルは参考にしない
- ロゴ・テキスト・ボタンの中央寄せ・余白は現状のバランスを維持したまま、囲みの箱だけを取り除く
